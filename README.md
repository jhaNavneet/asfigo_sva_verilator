Contains local copy of SVUnit + Verilator

Setup:

 * git remote add origin git@github.com:svenka3/af_cip_verilator
 * git pull origin main
 * git submodule update --init --recursive

To run the code:

  * cd vlt_svunit/svunit/
  * source Setup.csh
  * cd -
  * cd sim_dir
  * make
